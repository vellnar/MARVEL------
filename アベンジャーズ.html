THE AVENGERS

– S.H.I.E.L.D. TACTICAL OVERVIEW –

トニー・スターク / アイアンマン

億万長者 / 天才発明家 // チームの頭脳
最初はチームプレイを拒み、スティーブとも衝突するが、コールソン捜査官の死をきっかけに真に戦う意味を見出す。最新スーツ「マーク7」をまとい、核ミサイルを背負ってワームホールへ飛び込む決死の覚悟を見せる。
「天才、金持ち、プレイボーイ、博愛主義者。」

スティーブ・ロジャース / キャプテン・アメリカ

70年の眠りから覚めた英雄 // チームのリーダー
現代の複雑な世界とシールドの隠し事に困惑しつつも、ニューヨーク決戦では卓越した戦術眼で警察や他のメンバーへ見事な指揮を下す。誰もが認めるアベンジャーズの絶対的現場指揮官。
「鋼鉄の鎧を着た無敵の男。脱いだら何が残る？」

ソー / 雷神

アスガルドの神 // 義弟を追う者
地球を支配しようとする義弟ロキを止め、四次元キューブを回収するために雷鳴と共に再び地球へ降臨。ロキを心から憎みきれず苦悩するが、彼の凶行を止めるためアベンジャーズと共に最前線で激突する。
「ロキはアスガルドの人間であり、私の弟だ！……いや、確かにここ数日で80人殺したが、彼は養子なのだ。」

ロキ / 悪戯の神

地球の侵略者 // チタウリを率いる黒幕
未知の種族「チタウリ」と手を結び、地球の王になるために四次元キューブ（テッセラクト）を強奪。セプター（杖）の力でクリント/ホークアイやセルヴィグ博士の精神を支配し、ヘリキャリアを襲撃、地球を未曾有の混乱に陥れる。
「お前たちは生まれながらに支配されるのを望んでいる。自由という名の嘘から解放してやろう。」
